# Rubrica Telefonica Swing
Applicazione desktop per la gestione di contatti telefonici, sviluppata in Java con la libreria grafica Swing.
## Branch main

Versione base dell'applicazione:
* **Nuovo:** Aggiungi un contatto nuovo alla rubrica.
* **Modifica:** Modifica un contatto presente in rubrica.
* **Elimina:** Elimina un contatto presente in rubrica.
* **Interfaccia grafica:** Realizzata con Java Swing.
* **Gestione persistenza:** Salvataggio e caricamento dati su File "informazioni.txt".
